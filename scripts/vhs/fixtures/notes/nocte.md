# nocte

`nocte` is a small keyboard-first note app for quick local Markdown notes.

## Why it exists

- Keep capture fast in the terminal
- Store notes as plain files
- Stay local-first and easy to inspect

## Daily flow

1. Type a note name in the launcher.
2. Open an existing note or create a new one.
3. Edit in plain text and preview Markdown when needed.

## Useful commands

- `:list` shows every note sorted by recent updates
- `:export-all` rebuilds the HTML export folder
- `:files` opens the notes directory in the system file manager

## Editor shortcuts

- `Ctrl+P` toggles the live preview
- `Ctrl+E` exports the current note as HTML
- `Ctrl+L` opens the links dialog
- `Ctrl+D` deletes the current note after confirmation
